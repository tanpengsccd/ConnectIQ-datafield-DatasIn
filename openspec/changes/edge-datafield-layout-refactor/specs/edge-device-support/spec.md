## 新增需求

### 需求:Edge设备数据框尺寸

系统必须支持 Edge 码表常见的数据框尺寸：高度为屏幕的 1/5，宽度为屏幕的 1/2 或 100%。

#### 场景:数据框高度固定比例
- **当** 系统获取 `dc.getHeight()`
- **那么** 可用绘制高度为屏幕高度的 20%（1/5）

#### 场景:半宽数据框
- **当** 用户配置数据框宽度为半屏
- **那么** 可用绘制宽度为 `dc.getWidth() / 2`

#### 场景:全宽数据框
- **当** 用户配置数据框宽度为全屏
- **那么** 可用绘制宽度为 `dc.getWidth()`

### 需求:小屏设备兼容

系统必须兼容 Edge 520/530/540 等小屏设备，显示效果不能溢出或失真。

#### 场景:小屏设备正常显示
- **当** 在 Edge 520/530/540 上运行
- **那么** 所有字段必须在数据框边界内正确显示

### 需求:大屏设备兼容

系统必须兼容 Edge 1030/1040/1050/150 等大屏设备，充分利用更大显示区域。

#### 场景:大屏设备正常显示
- **当** 在 Edge 1030/1040/1050/150 上运行
- **那么** 所有字段必须在数据框边界内正确显示
