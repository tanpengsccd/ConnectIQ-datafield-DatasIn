# ConnectIQ 栏位数据 插件

## 功能
在一个栏位中显示 1~4个数据项目

## 可选择的栏位数据
- 功率: 当前、3s平均,单位‘w’
- 坡度: 当前、最大坡度,单位‘%’
- 速度: 当前、最大速度、平均，单位‘kph’
- 心率: 当前、最大、平均, 单位'bpm'
- 齿轮: 前齿数、后齿数、前后比，单位'T'

数据优先来自于 码表原始数据
