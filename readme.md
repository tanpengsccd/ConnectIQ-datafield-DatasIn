# ConnectIQ 栏位数据 插件

## 功能
在一个栏位中显示 1~4 个数据项目，支持 Edge 码表设备。

## 已实现功能

### 核心布局 ✅
- [x] 1字段：居中大字体显示
- [x] 2字段：上下布局，支持配置决定大小顺序
- [x] 3字段：品字布局（上2小下1大 或 上1大下2小），支持配置
- [x] 4字段：田字 2×2 均匀分布
- [x] 数据格式：数值 + 单位在同一行（单位更小更淡）

### 数据显示 ✅
- [x] 功率 (Power)：整数，最大 9999w
- [x] 速度 (Speed)：整数，最大 99kpm
- [x] 心率 (HeartRate)：整数，最大 999bpm
- [x] 坡度 (Grade)：小数点后1位，单位 %

### 设备兼容 ✅
- [x] 大屏设备自适应（Edge 1030/1040/1050/150）
- [x] 小屏设备自适应（Edge 520/530/540）
- [x] 移除手表遮挡模式，专注码表场景

## 待完成功能

### Bug 修复
- [ ] **数字跳动问题**：1位变2位时，单位会往右跳动
  - 原因：数值宽度变化导致整体偏移
  - 解决方案：使用等宽数字字体，或固定数值区域宽度

- [ ] **单位对齐问题**：单位应该和数字下底边对齐，而不是水平居中
  - 当前：单位与数值水平居中对齐
  - 期望：单位底部与数值底部对齐
  - 数据栏高度较窄时 则 上下数向中心靠拢显示
  - 数据栏高度较窄时 则 左右数向中心靠拢显示
  

### 功能完善
- [ ] **坡度算法**：当前无数据源，需自己实现坡度计算
  - 可基于 GPS 海拔数据计算
  - 基于 气压数据计算

- [ ] **齿数/踏频显示**：未实现
  - 前齿数、后齿数、前后比 
  - 单位 'T'（齿数）
  

### 设备兼容
- [ ] **Edge 520 兼容测试**：需要实际设备测试验证

## 可选择的栏位数据
- 功率: 当前、3s平均, 单位 'w'
- 坡度: 当前、最大坡度, 单位 '%'
- 速度: 当前、最大速度、平均, 单位 'kpm'
- 心率: 当前、最大、平均, 单位 'bpm'
- 齿轮: 前齿数、后齿数、前后比, 单位 'T'

数据优先来自于码表原始数据。

## 技术实现
- 布局方式：代码绘制（dc.drawText），动态计算位置
- 字体选择：根据屏幕高度自动选择 FONT_TINY/FONT_MEDIUM/FONT_LARGE
- 数据更新：compute() 接收传感器数据，onUpdate() 负责绘制
